# Customer Churn Prediction

## Project Overview
This project uses Machine Learning to predict whether a customer is likely to leave a company (churn) or continue using its services.
The model analyzes customer information such as demographics, account details, and banking behavior to identify churn patterns.

## Objective
Build a Machine Learning model that predicts customer churn and helps businesses improve customer retention strategies.

## Dataset
Dataset File:
* Churn_Modelling.csv
Target Variable:
* Churn
Values:
* 0 = Customer Stays
* 1 = Customer Leaves

## Technologies Used
* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Steps Performed
1. Imported required libraries.
2. Loaded the dataset.
3. Performed Exploratory Data Analysis (EDA).
4. Checked missing values.
5. Analyzed churn distribution.
6. Removed unnecessary columns.
7. Encoded categorical variables.
8. Split the dataset into training and testing sets.
9. Trained a Logistic Regression model.
10. Evaluated model performance.

## Evaluation Metrics
* Accuracy Score
* Classification Report
* Confusion Matrix
